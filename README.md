## 下载

### Docker

```
docker pull apachecn0/geeksforgeeks-dsal-zh
docker run -tid -p <port>:80 apachecn0/geeksforgeeks-dsal-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g geeksforgeeks-dsal-zh
geeksforgeeks-dsal-zh <port>
# 访问 http://localhost:{port} 查看文档
```
