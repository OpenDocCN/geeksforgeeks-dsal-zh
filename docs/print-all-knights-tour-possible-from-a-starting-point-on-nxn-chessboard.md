# 在 NxN 棋盘

上打印所有可能的骑士之旅起点

> 原文:[https://www . geeksforgeeks . org/print-all-knights-tour-从一个可能的起点出发-on-nxn-棋盘/](https://www.geeksforgeeks.org/print-all-knights-tour-possible-from-a-starting-point-on-nxn-chessboard/)

给定一个 **N x N** 棋盘，一个骑士最初站在 **X <sup>第</sup>T5】行和 **Y <sup>第</sup>T9】列上，任务是打印所有可能的路径，这样骑士必须精确地访问每个方块一次。****

**示例:**

> **输入:** N = 5，X = 1，Y = 1
> **输出:**
> 1 6 15 10 21
> 14 9 20 5 16
> 19 2 7 22 11
> 8 13 24 17 4
> 25 18 3 12 23
> 1 6 11 18 21
> 12 17 20 5 10
> 7 2 15 22 19【T13
> 
> ……302 更
> **说明:**最初，骑士在(1，2) <sup>第</sup>号牢房。按照第 1 条路径，骑士将按照以下顺序访问细胞:(1，1)-(T5】(3，2)-(T6】(5，3)-(T7】(4，5)-(T8】(2，4)……以此类推。
> 
> **输入:** N = 3，X = 1，Y = 3
> **输出:** -1
> **解释:**不存在有效的路径序列，使得骑士只访问每个广场一次。

**方法:**通过[递归](https://www.geeksforgeeks.org/recursion/)和[回溯](https://www.geeksforgeeks.org/backtracking-algorithms/)的帮助，逐一生成所有可能的行程并检查是否满足给定条件，可以解决问题。在[骑士之旅问题](https://www.geeksforgeeks.org/the-knights-tour-problem-backtracking-1/)中讨论了类似方法的更彻底的解释。以下是要遵循的步骤:

*   创建一个[递归函数](https://www.geeksforgeeks.org/recursive-functions/)来迭代骑士可以遵循的所有可能路径。
*   使用变量**保持骑士访问的方块数**。
*   创建一个函数 **isSafe()** ，该函数以一个正方形的坐标为参数，返回该正方形对于下一个骑士的移动是否有效。
*   迭代骑士所有可能的 8 个动作，检查它们是否可以安全访问。如果是，递归调用下一步，直到访问的方块数等于方块总数。

下面是上述方法的实现:

## C++

```
// C++ program of the above approach
#include <bits/stdc++.h>
using namespace std;

// Stores the 8 possible combinations of
// moves that the knight can follow
int DirX[] = { 2, 1, -1, -2, -2, -1, 1, 2 };
int DirY[] = { 1, 2, 2, 1, -1, -2, -2, -1 };

// Function to find if (i, j) is a valid
// cell for the knight to move and it
// exists within the chessboard
bool isSafe(int i, int j, int n,
            vector<vector<int> >& Board)
{
    return (i >= 0 and j >= 0 and i < n and j < n
            and Board[i][j] == 0);
}

// Stores whether there exist any valid path
bool isPossible = false;

// Recursive function to iterate through all
// the paths that the knight can follow
void knightTour(vector<vector<int> >& ChessBoard, int N,
                int x, int y, int visited = 1)
{
    // Mark the current square of the chessboard
    ChessBoard[x][y] = visited;

    // If the number of visited squares are equal
    // to the total number of sqares
    if (visited == N * N) {
        isPossible = true;

        // Print the current state of ChessBoard
        for (int i = 0; i < N; i++) {
            for (int j = 0; j < N; j++) {
                cout << ChessBoard[i][j] << " ";
            }
            cout << endl;
        }
        cout << endl;

        // Backtrack to the previous move
        ChessBoard[x][y] = 0;
        return;
    }

    // Iterate through all the eight possible moves
    // for a knight
    for (int i = 0; i < 8; i++) {

        // Stores the new position of the knight
        // after a move
        int newX = x + DirX[i];
        int newY = y + DirY[i];

        // If the new position is a valid position
        // recursively call for the next move
        if (isSafe(newX, newY, N, ChessBoard)
            && !ChessBoard[newX][newY]) {
            knightTour(ChessBoard, N, newX, newY,
                       visited + 1);
        }
    }

    // Backtrack to the previous move
    ChessBoard[x][y] = 0;
}

// Driver Code
int main()
{
    vector<vector<int> > ChessBoard(5, vector<int>(5, 0));
    int N = ChessBoard.size();
    int X = 1;
    int Y = 1;

    knightTour(ChessBoard, N, X - 1, Y - 1);

    // If no valid sequence of moves exist
    if (!isPossible) {
        cout << -1;
    }

    return 0;
}
```
