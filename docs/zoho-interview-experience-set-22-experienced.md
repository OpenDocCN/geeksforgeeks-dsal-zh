只有用户定义的函数。完全没有内置功能。
不扫描输入，直接分配给 vars。

第一级:(2 小时–纸笔)
1。用给定值求解方程 X 次方 Y。

```
Example:
    Input: X=2, Y=-2
    Output: 0.25
```

2.找出两个给定点之间的距离，并四舍五入到最接近的数字。

```
Example:
    Input: (2,4)(4,10)
    Output: 6
```

3.计算处理“{content}”的给定字符串中的字符数。作为转义序列。if“{ content }”前面是“”，将其视为正常的“{ content }”；而不是转义序列。如果“发生了，就把它当作单一的”。

```
Example:
    Input: Hello$World$
    Output: 11
```

4.给定一个 2D 矩阵，求所有元素的和。

```
Example:
    Input: [1 2 3]
           [4 5 6]
           [7 8 9]
    Output: 45
```
