****2.我们有 n 张牌，每张牌从 1 到 n 编号。
所有牌都是随机洗牌的。
我们只允许操作 MoveCard(n)，将值为 n 的卡片移动到堆的顶部。
我们需要用最少的 MoveCard 操作次数对一堆卡片进行排序。
我能想到的最天真的方法是从 MoveCard(n)、MoveCard(n-1)、MoveCard(n-2)开始……移动卡(1)。
这种方法将解决 n 个 MoveCard 操作中的问题。
