# 检查是否可以通过置换 A 的二进制数字形成 B

> 原文:[https://www . geesforgeks . org/check-if-b-可以通过置换 a 的二进制数字来形成/](https://www.geeksforgeeks.org/check-if-b-can-be-formed-by-permuting-the-binary-digits-of-a/)

给定两个整数 **A** 和 **B** ，任务是检查 **B** 的二进制表示是否可以通过置换 **A** 的二进制数字来生成。
**举例:**

> **输入:** A = 3，B = 9
> **输出:**是
> 二进制(3) = 0011 和二进制(9) = 1001
> **输入:** A = 6，B = 7
> **输出:**否

**方法:**想法是计算两个数的二进制表示中的设定位数，现在如果它们相等，那么答案是肯定的，否则答案是否定的
下面是上述方法的实现:

## C++

```
// C++ implementation of the approach
#include <bits/stdc++.h>
using namespace std;

// Function that returns true if the
// binary representation of b can be
// generated by permuting the
// binary digits of a
bool isPossible(int a, int b)
{

    // Find the count of set bits
    // in both the integers
    int cntA = __builtin_popcount(a);
    int cntB = __builtin_popcount(b);

    // If both the integers have
    // equal count of set bits
    if (cntA == cntB)
        return true;
    return false;
}

// Driver code
int main()
{
    int a = 3, b = 9;

    if (isPossible(a, b))
        cout << "Yes";
    else
        cout << "No";

    return 0;
}
```
