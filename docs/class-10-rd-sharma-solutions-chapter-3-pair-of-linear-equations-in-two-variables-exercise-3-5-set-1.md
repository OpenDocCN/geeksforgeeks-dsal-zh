### 问题 1。在下面的每一个方程组中，确定该系统是否有唯一解、无解或无穷解。如果有独特的解决方案:

### x 3y 3 = 0，

### 3x 9y 2 = 0

**解决方案:**

> 鉴于此，
> 
> x 3y 3 = 0…(1)
> 
> 3x 9y 2 = 0…(2)
> 
> 所以，给定的方程的形式是:
> 
> a1x+b1y C1<sub>= 0…(3)</sub>
> 
> a2x+b2y–C2 = 0…(4)
> 
> 通过比较等式(1)和等式(3)以及等式(2)和等式(4)，我们得到
> 
> a1 = 1，B1 = 3，C1 = 3
> 
> a2 = 3，B2 = 9，C2 = 2
> 
> 让我们检查一下等式，
> 
> a1/a2 = 1/3
> 
> b1/b2 = -3/-9 = 1/3
> 
> c1/c2 = -3/-9 = 3/2
> 
> a1/a2 = b1/b2 ≠ c1/c2
> 
> **因此，给定的方程组没有解。**
